# CRM Frontend (React + Tailwind + Firebase Auth client)

Features:

- Login / Register (Firebase client)
- Dashboard, Leads, Users, Settings pages
- Leads: list, add, edit, delete (connected to backend /api/leads)
- Tailwind CSS, Vite

Setup:

1. npm install
2. Create .env with VITE*API_BASE_URL and VITE_FIREBASE*\* keys
3. npm run dev
